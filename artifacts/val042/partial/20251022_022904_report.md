# Section 3: Business Analysis

## S3.1: Profitability Analysis

| Field | Answer |
| :---- | :---- |
| Revenue & Direct-Cost Dynamics | 公司2022年至2024年營收呈現穩健增長，分別為219,954,948千、276,744,954千及337,591,576千，2022→2023年增長約25.8%，2023→2024年增長約22.0%。毛利率由2022年的異常高值171.92%下降至2023年的35.12%，再提升至2024年的38.44%，顯示毛利率在2023年調整後趨於正常且持續改善。按產品/服務分解，配送服務營收由70,063,908千增至98,065,260千，佣金由56,509,699千增至95,340,956千，均顯著增長；在線營銷服務及其他服務亦呈穩定增長趨勢，反映多元化產品線均有良好表現。地理區域方面，營收全部來自中國市場，且持續增長，2022年219,954,948千增至2024年337,591,576千，顯示公司在中國市場的深耕與擴張成效顯著。整體而言，營收與直接成本動態反映公司營運規模擴大且毛利率穩健提升，產品多元且市場集中度高。 |
| Operating Efficiency | 營業利潤率由2022年的負2.65%大幅提升至2023年的4.85%，並進一步提升至2024年的10.91%，顯示營運效率顯著改善。營業收入與營收的比例由2022年負值轉為正值，2024年營業收入為36,844,956千，約占營收的10.91%，反映公司營運獲利能力增強。營業費用從2022年的70,257,787千增加至2024年的95,758,039千，增幅低於營收增長，顯示成本管理有效，費用控制相對合理。整體營運效率提升主要得益於營收擴大帶來的規模效應及成本控制，促使營業利潤率持續改善。 |
| External & One-Off Impact | 有效稅率由2022年的負1.04%提升至2023年的1.17%，再大幅提升至2024年的5.73%，顯示公司稅務負擔逐年增加，可能反映稅務政策變化或利潤結構調整。非經常性項目或異常模式未明顯顯示，2022年毛利率異常高企可能為數據異常或會計調整所致。外部因素方面，公司營收及獲利均集中於中國市場，受中國經濟環境及政策影響較大，未見其他地區分散風險。整體而言，稅率上升對淨利潤有一定壓力，但公司獲利能力仍持續改善。 |
